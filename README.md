# Library Management System

A simple **Python library management system** that allows users to view available books, add them to a cart, and generate a bill. This project demonstrates basic Python programming concepts like dictionaries, loops, functions, and file handling.

---

## Features

- Display available books with prices
- Add books to cart with quantity
- Generate a detailed bill
- Save bill to a file (`bill.data`)
- Handles invalid inputs gracefully
